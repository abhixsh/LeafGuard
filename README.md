__# Leaf Algae Detection - AI-Powered Plant Disease Identification

The mobile application for **Leaf Algae Detection** uses AI to identify plant diseases, specifically algae infections, through leaf image analysis. Built with **Flutter** and powered by **TensorFlow** models trained on Google’s **Teachable Machine**, this app enables users to quickly diagnose plant health, providing them with useful feedback and treatment recommendations.

## Project Overview

**Leaf Algae Detection** is designed to assist gardeners, farmers, and plant enthusiasts by offering fast and accurate disease detection for plants. The app allows users to capture leaf images and analyze them using an AI model, which can detect issues such as algae infections. The results include disease status, confidence level, and suggestions for treatment. Past scans are stored for future reference in the **History** section.

This repository contains the source code for the Flutter-based mobile application.

---

## Technologies Used

- **Flutter**: Cross-platform mobile app framework for building the frontend.
- **TensorFlow**: Machine learning framework for plant disease detection.
- **Teachable Machine**: Used for training the TensorFlow model to recognize plant diseases.
- **Provider**: State management for handling the app’s data (scan history, results, etc.).
- **Shared Preferences**: For storing scan history and results locally on the device.

---

## Features

- **AI-Driven Detection**: Identifies algae infections and other plant diseases based on leaf images.
- **Scan History**: Users can save past scan results, including leaf images, disease status, and scan dates.
- **Real-Time Scan**: Capture leaf images and get instant feedback on plant health.
- **Recommendations**: Provides treatment suggestions based on detected disease.
- **About Us**: Information about the app and the development team.

## Screenshots
